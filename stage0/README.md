# Stage 0 — Fuse Core Python interpreter

Run a Fuse file:

```bash
python src/main.py ../tests/fuse/milestone/four_functions.fuse
```

Check a Fuse file without running it:

```bash
python src/main.py --check ../tests/fuse/core/types/val_immutable.fuse
```

Run the Core test suite:

```bash
python tests/run_tests.py
```
